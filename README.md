# Quick Commerce Performance Dashboard 📊

## Overview
An interactive Power BI dashboard analyzing quick commerce 
performance across 6+ Indian cities.

## Key Metrics
- 948K total orders analyzed
- ₹541M total revenue
- 6+ cities covered
- 7 product categories

## Tools Used
Power BI | SQL | Excel | DAX | Data Visualization

## Dashboard Features
- Revenue by product category & city
- Delivery time trends by distance
- Customer rating impact on revenue
- Payment method distribution
- Key insights panel
